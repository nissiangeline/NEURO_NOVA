# **App Name**: Neuro Nova

## Core Features:

- Hero Section with Animation: Display an animated title and subtitle using SplitText with GSAP, creating a welcoming and visually appealing introduction.
- Gooey Navigation: Implement a fluid, interactive navigation bar using GooeyNav to enhance user experience.
- Visual Section with MasonryGrid: Showcase peaceful images using a MasonryGrid layout with GSAP animations for a calming and engaging visual experience.
- Myth vs Fact Flip Cards: Present educational content using interactive flip cards with rounded pastel styles and emoji accents.
- Voice Test Analysis: Allow users to record their voice and then analyze key characteristics of their voice using a tool such as an LLM. The LLM will then perform various analyses.
- Memory Game: Offer a memory game using pastel sequence buttons with Magic Bento interactivity to assess cognitive function.
- Typing Task Analysis: Implement a typing task that monitors and assesses typing behavior. Typing analyses can be performed by a tool such as an LLM to track metrics like hesitation and cohesion.
- Language Toggle: Provide a language toggle for multi-language support, enhancing accessibility for a diverse user base.

## Style Guidelines:

- Primary color: Maldives blue (#79D5FA) for calmness and trust, reminiscent of clear waters.
- Background color: Soft beige (#F5F5DC) to provide a neutral, soothing canvas that evokes feelings of warmth and serenity, promoting relaxation.
- Accent color: Blush peach (#F0B2A7) to add a touch of gentle vibrancy without being overwhelming. It can be used sparingly for highlights and interactive elements, enhancing engagement while maintaining the app's calming aesthetic.
- Font pairing: 'Poppins' (sans-serif) for headlines, and 'PT Sans' (sans-serif) for body text, for a contemporary yet readable feel.
- Note: currently only Google Fonts are supported.
- Use hand-drawn style icons for a friendly and approachable aesthetic.
- Implement smooth GSAP animations throughout the app to create fluid transitions and subtle visual feedback.
- Incorporate rounded corners and gentle shadows to soften the interface and add depth.
